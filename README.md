## HotelBooking.

## Purpose

This project is a modern Hotel Booking Platform that provides users with a seamless and engaging experience for discovering and booking hotel rooms.

## Live URL

[Live Demo](#) _(https://hotel-booking-19def.web.app/)_

## Key Features

- Responsive Design
- Log in and Sign up
- Book room
- cancel booking
- Add Review
